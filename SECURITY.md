# Security Policy

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Use GitHub's private vulnerability reporting by clicking **"Report a vulnerability"** on the Security tab of this repository. You will receive a response within 48 hours. If confirmed, a fix will be prioritised and a patched release issued.

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
