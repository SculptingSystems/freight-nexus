# Code of Conduct

We pledge to make participation in this project a harassment-free experience for everyone.

Expected behaviour: welcoming language, respectful disagreement, constructive criticism.

Unacceptable behaviour: harassment, trolling, personal attacks, publishing private information.

Report issues to **conduct@sculptingsystems.dev**.

Adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
